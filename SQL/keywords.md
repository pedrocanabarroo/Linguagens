# Glossário de Comandos SQL

| Palavra-chave (Keyword) | Descrição |
| :--- | :--- |
| **ADD** | Adiciona uma coluna em uma tabela existente. |
| **ADD CONSTRAINT** | Adiciona uma restrição após a tabela já ter sido criada. |
| **ALL** | Retorna verdadeiro se todos os valores da subquery atenderem à condição. |
| **ALTER** | Adiciona, exclui ou modifica colunas em uma tabela, ou altera o tipo de dados de uma coluna. |
| **ALTER COLUMN** | Altera o tipo de dados de uma coluna em uma tabela. |
| **ALTER TABLE** | Adiciona, exclui ou modifica colunas em uma tabela. |
| **ALTER VIEW** | Atualiza uma view. |
| **AND** | Inclui apenas linhas onde ambas as condições são verdadeiras. |
| **ANY** | Retorna verdadeiro se qualquer um dos valores da subquery atender à condição. |
| **AS** | Renomeia uma coluna ou tabela com um alias. |
| **ASC** | Ordena o conjunto de resultados em ordem crescente. |
| **BACKUP DATABASE** | Cria um backup de um banco de dados existente. |
| **BETWEEN** | Seleciona valores dentro de um intervalo dado. |
| **CASE** | Cria diferentes saídas baseadas em condições. |
| **CHECK** | Uma restrição que limita o valor que pode ser colocado em uma coluna. |
| **COLUMN** | Altera o tipo de dados ou exclui uma coluna em uma tabela. |
| **CONSTRAINT** | Adiciona ou exclui uma restrição. |
| **CREATE** | Cria um banco de dados, índice, view, tabela ou procedure. |
| **CREATE DATABASE** | Cria um novo banco de dados SQL. |
| **CREATE INDEX** | Cria um índice em uma tabela (permite valores duplicados). |
| **CREATE OR REPLACE VIEW** | Atualiza uma view. |
| **CREATE TABLE** | Cria uma nova tabela no banco de dados. |
| **CREATE PROCEDURE** | Cria um stored procedure (procedimento armazenado). |
| **CREATE UNIQUE INDEX** | Cria um índice único em uma tabela (não permite duplicados). |
| **CREATE VIEW** | Cria uma view baseada no resultado de um SELECT. |
| **DATABASE** | Cria ou exclui um banco de dados SQL. |
| **DEFAULT** | Restrição que fornece um valor padrão para uma coluna. |
| **DELETE** | Exclui linhas de uma tabela. |
| **DESC** | Ordena o conjunto de resultados em ordem decrescente. |
| **DISTINCT** | Seleciona apenas valores distintos (diferentes). |
| **DROP** | Exclui uma coluna, restrição, banco de dados, índice, tabela ou view. |
| **DROP COLUMN** | Deleta uma coluna em uma tabela. |
| **DROP CONSTRAINT** | Deleta uma restrição UNIQUE, PRIMARY KEY, FOREIGN KEY ou CHECK. |
| **DROP DATABASE** | Deleta um banco de dados SQL existente. |
| **DROP DEFAULT** | Deleta uma restrição DEFAULT. |
| **DROP INDEX** | Deleta um índice em uma tabela. |
| **DROP TABLE** | Deleta uma tabela existente no banco de dados. |
| **DROP VIEW** | Deleta uma view. |
| **EXEC** | Executa um procedimento armazenado. |
| **EXISTS** | Testa a existência de qualquer registro em uma subquery. |
| **FOREIGN KEY** | Chave usada para ligar duas tabelas entre si. |
| **FROM** | Especifica de qual tabela selecionar ou excluir dados. |
| **FULL OUTER JOIN** | Retorna todas as linhas quando há correspondência na tabela da esquerda ou direita. |
| **GROUP BY** | Agrupa o conjunto de resultados (usado com COUNT, MAX, MIN, SUM, AVG). |
| **HAVING** | Usado em vez de WHERE com funções de agregação. |
| **IN** | Verifica se o valor da coluna corresponde a qualquer valor em uma lista. |
| **INDEX** | Cria ou deleta um índice em uma tabela. |
| **INNER JOIN** | Retorna linhas que possuem valores correspondentes em ambas as tabelas. |
| **INSERT INTO** | Insere novas linhas em uma tabela. |
| **INSERT INTO SELECT** | Copia dados de uma tabela para outra. |
| **IS NULL** | Testa valores vazios. |
| **IS NOT NULL** | Testa valores não vazios. |
| **JOIN** | Une tabelas. |
| **LEFT JOIN** | Retorna todas as linhas da tabela à esquerda e as correspondentes da direita. |
| **LIKE** | Procura por um padrão específico em uma coluna. |
| **LIMIT** | Especifica o número de registros a serem retornados. |
| **NOT** | Inclui apenas linhas onde a condição não é verdadeira. |
| **NOT NULL** | Restrição que impede que uma coluna aceite valores nulos. |
| **OR** | Inclui linhas onde pelo menos uma das condições é verdadeira. |
| **ORDER BY** | Ordena o resultado em ordem crescente ou decrescente. |
| **OUTER JOIN** | Retorna todas as linhas quando há correspondência em uma das tabelas. |
| **PRIMARY KEY** | Identifica de forma única cada registro em uma tabela. |
| **PROCEDURE** | Um procedimento armazenado. |
| **RIGHT JOIN** | Retorna todas as linhas da tabela à direita e as correspondentes da esquerda. |
| **ROWNUM** | Especifica o número de registros retornados (específico de alguns bancos). |
| **SELECT** | Seleciona dados de um banco de dados. |
| **SELECT DISTINCT** | Seleciona apenas valores diferentes. |
| **SELECT INTO** | Copia dados de uma tabela para uma nova tabela. |
| **SELECT TOP** | Especifica o número de registros retornados no topo do resultado. |
| **SET** | Especifica as colunas e valores que devem ser atualizados. |
| **TABLE** | Cria, altera ou deleta tabelas ou dados dentro delas. |
| **TOP** | Especifica o número de registros retornados. |
| **TRUNCATE TABLE** | Deleta os dados de uma tabela, mas mantém sua estrutura. |
| **UNION** | Combina resultados de SELECTs (apenas valores distintos). |
| **UNION ALL** | Combina resultados de SELECTs (permite valores duplicados). |
| **UNIQUE** | Garante que todos os valores em uma coluna sejam únicos. |
| **UPDATE** | Atualiza linhas existentes em uma tabela. |
| **VALUES** | Especifica os valores de uma instrução INSERT INTO. |
| **VIEW** | Cria, atualiza ou deleta uma visualização. |
| **WHERE** | Filtra registros que cumprem uma condição específica. |
